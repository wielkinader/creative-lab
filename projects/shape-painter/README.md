# Shape Painter

Rebuilds an uploaded photo using only simple geometric shapes (circles, triangles,
rectangles), within a chosen shape budget, then plays the reconstruction back as a
timelapse.

Status: scaffold only. Algorithm, WebGL renderer and controls land in later milestones.

Full README (how it works, parameters, credits) is written in the final milestone.
Inspired by Michael Fogleman's [Primitive](https://github.com/fogleman/primitive).
